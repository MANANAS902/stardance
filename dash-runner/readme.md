# Pulse Runner

A Geometry Dash-inspired browser game built with HTML, CSS, and JavaScript.

## How to Play

- Open `index.html` in a browser.
- Press `Space`, `Up Arrow`, click, or tap to jump.
- Avoid the spikes and blocks for as long as you can.
- Use the speed slider to make the run easier or harder.
- Press `R` or use the restart button after a crash.

## Notes

- Best score is saved in the browser with `localStorage`.
- The game is fully playable as a static website; no build step is required.
