# Neon Dash

A Geometry Dash-inspired browser game built with HTML, CSS, and JavaScript.

## How to Play

- Open `index.html` in your browser.
- Press `Space`, `Up Arrow`, or click to jump.
- Avoid the obstacles for as long as you can.
- Use the difficulty slider to make the run faster or easier.
- Press `R` or use the restart button after a crash.

## Game Mechanics

- The game scrolls automatically and increases in intensity over time.
- Score is based on distance traveled and obstacles cleared.
- Best score is stored locally in the browser.
